# Setup
1. Install the package
   
    ```
    npm install brownsugar
    ```
2. Enable strict mode inside tsconfig.json
    
    ```json
    {
        "compilerOptions": {
            // ...
            "strict": true
        }
    }
    ```
    This will enable strict type checking which is required for correct usage of the library.
